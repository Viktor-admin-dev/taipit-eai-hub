## 1. Новая заявка с AI-превью

**Путь:** https://taipit.starec.ai/how-to-participate → "Подать заявку"

1. Заполни форму (имя, email, дивизион, тип и т.д.)
2. Заполни описание проблемы и решения (≥100 символов каждое)
3. Нажми **"🔍 Проверить заявку"** — должна появиться панель с:
   - Баллами по 5 критериям (0–100)
   - Зелёный блок "Что хорошо"
   - Жёлтый блок "Можно улучшить" с конкретными советами
   - Синий блок "Быстрые улучшения"
4. Нажми **"✓ Отправить заявку"** — заявка создаётся, AI-оценка запускается в фоне (~30 сек)
5. Убедись что появился экран "Заявка отправлена!" со ссылкой "Редактировать заявку позже"

**Что проверить в логах при новой заявке:**
```
docker logs taipit-eai-hub --tail 20
```
Должно быть: `Batch eval OK: application #N` или `Post-submit AI flow` без ошибок.

## 3. Просмотр AI-оценки в админ-панели

**Путь:** https://taipit.starec.ai/admin → Заявки → любая

1. Войди в админку (`eai-hub-admin-2026`)
2. Открой /admin/applications — список всех 8 заявок
3. Кликни на любую заявку (#1–#8)
4. В правой колонке должна быть AI-карточка:
   - Переключатель истории оценок (если несколько)
   - Вердикт (🟢 Поддержать / 🟡 Развить / 🔴 Переосмыслить)
   - Общий балл
   - One-liner
   - Профиль автора (AI-уровень, качества, зона роста)
   - Для Оксаны и Шиканова: проблема/решение простыми словами, бизнес-эффект
   - Скрытый потенциал
   - Как поддержать (тип менторинга, ментор, ресурсы)

## 4. Предпросмотр и отправка фидбека автору

На странице заявки в правой AI-карточке:

1. Нажми **"💬 Автору"**
2. Должен появиться модал с:
   - **От кого:** `eai@taipit.starec.ai`
   - **Кому:** email участника (зелёный)
   - **Тема:** редактируемое поле
   - **Предупреждение** (жёлтое): "RESEND_API_KEY не настроен — письмо не будет отправлено реально"
   - **Превью HTML** письма в iframe (градиентный заголовок, блоки с оценкой, советы)
3. Можно отредактировать тему письма
4. Нажми **"📋 Записать в лог (без отправки)"** — письмо записывается в EmailLog

> ⚠️ Реальная отправка заработает после настройки `RESEND_API_KEY` в docker-compose.yml

## 7. Пакетная оценка всех заявок

**Путь:** https://taipit.starec.ai/admin (главная страница дашборда)

1. Прокрути вниз до блока **"🤖 AI-оценка всех заявок"**
2. Нажми **"Запустить"**
3. Ответ: `Started batch evaluation for N applications`
4. Оценка идёт в фоне (~2 сек между заявками)
5. Проверь через 1–2 минуты: `docker logs taipit-eai-hub --tail 30`

> Если все заявки уже оценены — ответ: `All applications already evaluated`
